## Featured projects

### CipherVault

A deployed password-based privacy tool for locking and unlocking messages and files. It is designed for users who need to protect sensitive content without accounts, tracking, or external password storage.

- Live app: [ciphervault-55jh.onrender.com](https://ciphervault-55jh.onrender.com/)
- Stack: Python, Flask, PBKDF2, HTML, CSS, JavaScript

### Domain Enumerator

An attack-surface mapping tool that combines DNS brute forcing, passive OSINT, certificate-transparency queries, and concurrent HTTPS validation. It helps security and pentest teams reduce repetitive reconnaissance work and discover hosts that manual processes may miss.

- GitHub: [subdomain-enumerator](https://github.com/KhushiDhankhar/subdomain-enumerator)
- Stack: Python, DNS, multithreading, BeautifulSoup, crt.sh

### Network Utilities Suite

A consolidated toolkit for port scanning, hash lookup, and file-signature checks. It reduces tool switching for engineers while retaining structured, traceable audit output.

- GitHub: [network-utilitity-tools](https://github.com/KhushiDhankhar/network-utilitity-tools)
- Stack: Python, sockets, TCP/IP, Tkinter
