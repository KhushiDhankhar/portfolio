01 / LIVE DEPLOYMENT / PYTHON + FLASK

CipherVault

A simple password-based privacy tool for locking and unlocking messages and files. It is built for people who want straightforward privacy without accounts, tracking, or handing their passwords to a third party.

Why it exists

Sharing private content through everyday channels can require trusting platforms with sensitive information. CipherVault makes protection accessible before that content leaves the user's hands.

Who it helps

Privacy-conscious individuals who need to share a confidential message or file and want a small, understandable workflow rather than a closed, account-based service.

What I built

  • Flask-based web experience for four core actions: lock or unlock a message or file.
  • Password-driven protection flow with password-to-key derivation and local-first processing principles.
  • Deployed live application with a clear, no-recovery privacy model.

Impact

Users can protect text and files without an account or external password storage, giving them a practical alternative when they need privacy with fewer trust assumptions.

02 / SECURITY AUTOMATION / PYTHON

Domain Enumerator

An attack-surface mapping utility that unifies several discovery methods into a concurrent, configurable reconnaissance workflow.

Why it exists

Manual subdomain discovery is repetitive and leaves blind spots. Security work needs a repeatable way to combine active and passive sources before validating what is actually reachable.

Who it helps

Security and pentest teams mapping an organization's public-facing surface before deeper assessment.

What I built

  • Four-stage pipeline combining DNS brute forcing, Bing OSINT scraping, certificate-transparency queries, and HTTPS validation.
  • Configurable ThreadPoolExecutor concurrency and an argparse-driven command-line interface.
  • Modular workflow designed to scale across large wordlists.

Impact

Turns disconnected discovery steps into one concurrent workflow, reducing repetitive recon effort and surfacing hosts that a manual-only process can miss.

03 / SECURITY TOOLKIT / PYTHON

Network Utilities Suite

A consolidated toolkit for common network and file-security checks, created to reduce unnecessary context switching during routine technical work.

Why it exists

Separate utilities for port scanning, hash lookup, and file checks add friction to day-to-day investigation work and can make auditability inconsistent.

Who it helps

Engineers and security practitioners who need common diagnostic utilities with clear outputs and traceable activity.

What I built

  • Concurrent Python port scanner using sockets, custom ranges, and service identification.
  • CLI hashing utility with persistent lookup storage.
  • Tkinter file-signature scanner plus rotating logs and UUID session tracking.

Impact

Consolidates three common checks into one suite, helping users move faster while retaining structured audit information for later review.
